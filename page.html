PRO-C187: MAKE YOUR OWN PROJECT

Goal of the Project:

In this, you have to make your own AR project using pattern markers in A-Frame.

Getting Started:

1. Open VS Code Editor.
2. You should make your own project and yeah.

Submitting the Project:

1. Upload your completed project to your own GitHub account.
2. Create a new repository named Project 187.
3. Upload your project code to this GitHub repository.
4. Submit the published link of the project in the Student Dashboard.

Stealing the Project:

1. Scan the marker below.
2. Use everything on the marker to make your own project.
